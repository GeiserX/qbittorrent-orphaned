# CLAUDE.md — qbittorrent-orphaned

## Overview
Lightweight Python utility that identifies orphaned files on disk not tracked by any torrent in a qBittorrent instance. Connects to the qBittorrent Web API v2 to cross-reference every file against active torrents.

## Tech Stack
- Python 3.8+
- requests library (sole runtime dependency)
- Published on PyPI as `qbittorrent-orphaned`
- pytest for testing

## Development
```bash
pip install -e ".[dev]"
pytest
```

Run standalone:
```bash
QBIT_HOST=http://localhost:8080 QBIT_USER=admin QBIT_PASS=yourpassword \
CATEGORY_FOLDERS="Films=/mnt/media/films;Shows=/mnt/media/shows" \
python orphan_detector.py
```

## Architecture
- `orphan_detector.py` — main detection logic (single-file core)
- `tests/` — test suite
- `pyproject.toml` — packaging and metadata
- `docs/` — documentation and images
